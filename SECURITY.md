## Reporting a Vulnerability

We take the security of EnvSafe seriously. If you believe you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to [security@envsafe.dev](mailto:security@envsafe.dev).

You should receive a response within 48 hours. If for some reason you do not, please follow up with us to ensure we received your original message.
